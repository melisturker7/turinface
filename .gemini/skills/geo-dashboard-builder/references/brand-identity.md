# Brand Identity & UI Standards

## Color Palette
- **Background**: `#0a1628` (Deep Midnight)
- **Primary**: `#8DC63F` (Eco Green)
- **Accent**: `#00A7ED` (Deep Sky Blue)
- **Danger**: `#B22025` (Warning Red)
- **Warning**: `#FFC500` (Safety Yellow)
- **Card/Sidebar**: `#0f2240` / `#071222`
- **Border**: `#1e4976`

## Typography
- **Primary**: System UI, Inter, or Sans-Serif.
- **Header**: Bold, uppercase for labels, 0.05em letter spacing.

## UI Components
- **Badges**: Rounded-full, semi-transparent background of the semantic color.
- **Map**: Always use "Stadia Alidade Smooth Dark" tiles for consistent aesthetic.
- **Sidebar**: Fixed width (350px) with scrollable event list.
